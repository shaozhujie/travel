# SpringBoot+Vue的旅游信息推荐系统

#### Description
系统角色上分为管理员以及普通用户进行实现，管理员主要负责整个网站后台的维护管理，例如包括用户管理、景点以及分类管理、旅游路线管理、酒店管理等功能；前台包括用户登陆注册、旅游景点查询和预定、旅游路线查询和收藏、酒店查询和预订等功能。
